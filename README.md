# 🌙 تطبيق إسلاميات | Islamiyat App

> تطبيق إسلامي شامل ومتقدم، يتميز بتصميم عصري واجهة مستخدم احترافية (UI/UX) لتقديم تجربة مريحة وسهلة للمستخدم المسلم.

![Islamiyat App Preview](https://picsum.photos/seed/islamiyat/800/400?blur=2)

## ✨ المميزات الرئيسية (Key Features)

*   📖 **القرآن الكريم (Holy Quran):**
    *   قراءة وتلاوة القرآن الكريم بخطوط واضحة وجميلة (الرسم العثماني).
    *   الاستماع لأشهر القراء بجودات متعددة.
    *   تتبع الآيات أثناء التلاوة (Active Ayah Highlighting) مع مؤثرات بصرية ذكية.
    *   مشغل صوتي متقدم مع إمكانية البث (Cast) والتحكم بالسرعة مع أمواج صوتية تفاعلية.
*   🕌 **مواقيت الصلاة واتجاه القبلة (Prayer Times & Qibla):**
    *   أوقات دقيقة للصلاة بناءً على الموقع الجغرافي.
    *   بوصلة تفاعلية لتحديد اتجاه القبلة بتصميم حديث وأنيق.
    *   عد تنازلي للصلاة القادمة والمتبقي لها.
*   🤲 **الأذكار والأدعية (Azkar & Supplications):**
    *   مكتبة شاملة لأذكار الصباح، المساء، بعد الصلاة، وغيرها من الأذكار اليومية.
    *   تصميم بطاقات تفاعلي يسهل القراءة، التنقل، والعد (سبحة إلكترونية مضمنة).
*   📚 **الحديث الشريف (Hadith):**
    *   باقة من الأحاديث النبوية الشريفة وتصنيفات متنوعة مع عرض للرواة والسند.
*   📻 **إذاعة القرآن الكريم (Islamic Radio):**
    *   بث مباشر لمحطات وإذاعات القرآن الكريم على مدار الساعة.
    *   مشغل راديو مدمج يعمل في الخلفية بسلاسة أثناء تصفح باقي التطبيق.

## 🎨 التخصيص والمظهر (Customization & Themes)

*   **الوضع الليلي والنهاري (Dark/Light Mode):** دعم كامل للتبديل السلس بين الأوضاع لحماية العين خصوصاً في أوقات الليل.
*   **خطوط محترفة (Typography):** خيارات متعددة للخطوط العربية لتناسب ذوقك (Cairo, Tajawal, Almarai, Amiri).
*   **تصميم متناسق (Harmonic Design):** تم بناء واجهات المستخدم بعناية فائقة باستخدام لوحات ألوان تعكس الطابع الإسلامي الأصيل بلمسة عصرية.
*   **دعم اللغتين (Bilingual Integration):** دعم كامل ومتقن للواجهات باللغتين العربية (من اليمين لليسار RTL) والإنجليزية (LTR).
*   **مؤثرات بصرية وتفاعلية:** خلفيات دايناميكية، حواف ناعمة، وانتقالات مرنة مبنية على أحدث معايير تصميم وتجربة المستخدم (UI/UX).

## 🚀 التقنيات المستخدمة (Tech Stack)

تم بناء التطبيق باستخدام أحدث مكتبات وتقنيات الويب الدقيقة لضمان وتوفير أفضل أداء واستجابة ممكنة:

*   **React 18+** ⚛️ - لبناء واجهات مستخدم تفاعلية.
*   **TypeScript** 📘 - لضمان جودة كتابة الكود والتخلص من الأخطاء البرمجية.
*   **Tailwind CSS** 🎨 - لتصميم الواجهات بطريقة المتجاوبة واحترافية.
*   **Vite** ⚡ - كبيئة تشغيل وبناء سريعة لتطبيقات الويب الحديثة.
*   **Framer Motion** 🔄 - للرسومات المتحركة (Animations) والانتقالات السلسة.
*   **Lucide Icons** ✨ - للحصول على أيقونات واضحة، دقيقة ومتناسقة الأوزان.

## 📱 التصميم المتجاوب (Responsive Design)

تم تصميم وتطوير التطبيق وفق مبدأ (Mobile-First) ليكون في متناول يدك على جميع الشاشات والأجهزة:
*   مظهر وتجربة مشابهة للتطبيقات الأصلية (Native App-like Experience) على الهواتف الذكية.
*   استغلال مثالي للمساحات والمناطق على الشاشات الكبيرة واللوحية (Tablets & Desktop).
*   تفاعلات متطورة عند اللمس (Touch-friendly Feedback) للأزرار والمكونات.

## ⚙️ البدء السريع (Getting Started)

لتشغيل المشروع وإعداده على بيئتك المحلية:

1.  **استنساخ المستودع (Clone the repo):**
    ```bash
    git clone https://github.com/your-username/islamiyat-app.git
    cd islamiyat-app
    ```

2.  **تثبيت الحزم (Install dependencies):**
    ```bash
    npm install
    ```

3.  **تشغيل خادم التطوير (Run dev server):**
    ```bash
    npm run dev
    ```

4.  قم بزيارة `http://localhost:3000` العرض المحلي في متصفحك.
